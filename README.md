# sylvine

Wprawka projektowa dot. dylematu więźnia rozgrywanego w turach pomiędzy wieloma graczami.

---

### how to build config

```
$ mkdir -p source/build && cd source/build
$ cmake .. && make install
$ cd ../..
$ ./source/build/prog
```
